내가 어디 있는지(경로) 알아야 한다 

경로의 핵심 개념 

루트(root) 디렉토리(/)
건물의 정문 또는 1층 로비, 모든 주소의시작점

홈 디렉토리(~)
나의 집 또는 개인 사무실 터미널을 처음 켰을 때 사용하는 작업 공간

절대 경로(absolute path)
정문 (루트, root, /) 부터 목적지까지의 전체 주소    ->   위치가 변하지 않는 설정파일
누가 어디서 보든 항상 똑같은 주소

상대 경로(relative path)                                            
현재 내 위치를 기준으로 한 주소
저 방으로 가(Desktop/) 또는 이전 방으로 가(..) 와 같음 

터미널이 운영체제마다 다름 
ex) macOS 리눅스 유닉스 계열 운영체제- bash
예외적 Windows -cmd

세상 모든 서버가 유닉스 기반
